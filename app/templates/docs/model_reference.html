Como funciona uma opção?
Nome exibido na lista
É o texto que aparece na lista suspensa do formulário. Também vai para a tabela de resumo do contrato (primeira página).
Texto completo para as cláusulas
Texto que aparece nas cláusulas do contrato. Se deixar em branco, o sistema usa o nome exibido no lugar — exceto em "Detalhamento do serviço", que ficará vazio.
O que o sistema faz sozinho
O sistema gera automaticamente o identificador interno de cada opção. Você não precisa se preocupar com isso.
